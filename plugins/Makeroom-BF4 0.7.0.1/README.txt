Makeroom-BF4
============

Makeroom Plugin for Battlefield 4 (BF4)

Created by Dan Caldwell (www.cidclan.net)

TO INSTALL OVER OLD VERSION
================================================================
To install update, shutdown your procon layer.
Remove the old "CMakeroom.DLL" and "CMakeroom.cs" in your plugins/BF4 directory.
Copy new CMakeroom.cs (from ZIP file) to plugins/BF4 directory.
Start your procon layer.
================================================================

FOR FIRST TIME INSTALLERS
================================================================
Shutdown your procon layer.
Copy "CMakeroom.cs" (from ZIP file) to plugins/BF4 directory.
Start your procon layer.
================================================================
